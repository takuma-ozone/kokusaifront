# :green_book: UI Components

## このセクションで説明すること

- UI コンポーネントのカタログを提供します。
- 各コンポーネントの記述方法・渡せるプロパティ・発火イベント等についてまとめています。

## コンポーネント一覧

:::tip フォーム部品の FW 定義について

フォーム部品は、バリデーション定義等を組み込むために Element UI を高速開発 FW としてラップしています。

これらのフォーム部品は`rm-〇〇`という名前で定義されており、デフォルトでは「ラベル＋フォーム部品」のセットとなっています。  
`has-label`プロパティに`false`を渡すことでラベルなしのコンポーネントを表示することが可能です。

:::
